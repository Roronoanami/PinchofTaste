🍽️ Pinch of Taste

A full-stack food ordering platform built using React, Spring Boot, and MongoDB. The application allows customers to browse food items, place orders, and manage their accounts while providing management features for food sellers.



Live Website:
https://pinchof-taste.vercel.app/

🚀 Features

Customer Features

- User Registration & Login
- Browse Food Menu
- View Food Details
- Add Items to Cart
- Place Orders
- Responsive User Interface

Seller Features

- Add Food Items
- Update Food Details
- Delete Food Items
- Manage Food Listings
- View Orders

🛠️ Tech Stack

Frontend
